Queens Game Guide

📖 About

Queens is a logic puzzle where you place exactly one queen in each row, column, and colored region of a grid. The catch? Queens cannot touch each other — not even diagonally! Inspired by Star Battle and popularized by LinkedIn, this addictive puzzle tests your logical deduction skills with grid sizes from 5×5 (beginner) to 12×12 (master). Every puzzle has a unique solution that can be found through pure logic — no guessing required!

🎯 Objectives

Place exactly one queen in each row, each column, and each colored region. Queens must never touch each other, including diagonally. Use logical deduction to find the unique valid arrangement.

🎮 How to Play

Click cells to place queens. Each row, column, and colored region needs exactly 1 queen. Queens cannot touch diagonally. Use X markers to track impossible cells.

⌨️ Controls

Click a cell to cycle: Empty → 👑 Queen → ✕ Marker → Empty. Right-click to cycle in reverse. Use markers to note cells where queens cannot go.

⚙️ Game Mechanics

  • ▸ Each row must contain exactly 1 queen
  • ▸ Each column must contain exactly 1 queen
  • ▸ Each colored region must contain exactly 1 queen
  • ▸ Queens cannot touch horizontally, vertically, or diagonally
  • ▸ Every puzzle has exactly one valid solution

💡 Tips

  • ✓ Start with the smallest regions — they have the fewest valid queen positions
  • ✓ Place an X marker on every cell diagonally adjacent to a placed queen
  • ✓ If a row or column has only one available cell, the queen must go there
  • ✓ Work through constraints systematically rather than guessing

❓ Frequently Asked Questions

What is the Queens game?
Queens is a logic puzzle where you place exactly one queen in each row, column, and colored region of a grid. Queens cannot touch each other, even diagonally. It's similar to Star Battle but with exactly one piece per unit instead of two.
Is Queens free to play?
Yes! Queens is completely free to play online. No download, registration, or payment required. Just pick a grid size and start solving puzzles instantly.
What grid size should I start with?
Start with the 5×5 grid to learn the basic rules and strategies. Once comfortable, progress to 7×7 and 8×8. Larger grids (10×10, 11×11, 12×12) offer increasingly complex challenges for experienced players.
How is Queens different from Star Battle?
Queens uses exactly 1 piece per row/column/region, while Star Battle uses 2. This makes Queens more accessible for beginners while still offering deep challenges at larger grid sizes. The diagonal touching rule is the same in both.
Do I need to guess to solve puzzles?
No! Every Queens puzzle has a unique solution that can be found through pure logical deduction. If you're guessing, try looking for more constrained cells first — especially in small regions or nearly-complete rows.
Can I play Queens on mobile?
Yes! Queens is fully responsive and works on phones, tablets, and desktops. Touch controls make it easy to place queens and markers on any device.
