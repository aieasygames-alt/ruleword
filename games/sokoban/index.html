Sokoban Game Guide

📖 About

Sokoban (Japanese for 'warehouse keeper') is the classic puzzle game that has challenged logical thinkers since 1982! Navigate through warehouse mazes, pushing boxes onto designated target locations. The catch? You can only push boxes, never pull them. One wrong move can create an impossible situation, requiring careful planning and spatial reasoning. This simple concept has spawned thousands of challenging levels!

🎯 Objectives

Push all boxes onto the target spots (marked with X or dots). Plan your route carefully—boxes can only be pushed, not pulled, and getting a box stuck in a corner or against a wall means restarting!

🎮 How to Play

Use arrow keys or WASD to move the player. Push boxes by walking into them. Get all boxes onto the target spots to complete each level. Boxes can only be pushed, never pulled.

⌨️ Controls

Use arrow keys (↑ ↓ ← →) or WASD to move. Walk into a box to push it one space in the same direction. Use Undo to take back moves or Reset to restart the level.

⚙️ Game Mechanics

  • ▸ Player can push one box at a time (not multiple)
  • ▸ Boxes can only be pushed, never pulled
  • ▸ Cannot push a box into a wall or another box
  • ▸ Level complete when all boxes are on target spots
  • ▸ A stuck box (corner, dead end) may require restart

✨ Features

  • ★ Multiple Levels - Increasing difficulty
  • ★ Undo System - Take back mistakes
  • ★ Reset Button - Start level over
  • ★ Move Counter - Track your efficiency

💡 Tips

  • ✓ Never push a box into a corner - it cannot be retrieved
  • ✓ Think several moves ahead before pushing a box
  • ✓ Use the undo feature to experiment without restarting
  • ✓ Some levels require pushing boxes in a specific order

❓ Frequently Asked Questions

Why can't I push this box?
You can only push one box at a time. If there's a wall or another box behind it, the push is blocked. Also check if you're trying to pull—you can only push!
What makes a level unsolvable?
If a box is pushed into a corner or against a wall where it can't reach any target, the level becomes unsolvable. Use undo or restart.
Is there always a solution?
Yes, all well-designed Sokoban levels have solutions. If you're stuck, try a different approach—you might need to push boxes in a specific order.
What's the best strategy?
Work backwards mentally—where does each box need to end up? Then figure out the order to push them. Avoid creating dead ends.
