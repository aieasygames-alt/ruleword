Killer Sudoku Game Guide

📖 About

Killer Sudoku combines the logic of Sudoku with the arithmetic challenge of Kakuro! Fill the grid following standard Sudoku rules, but also ensure that numbers in each outlined 'cage' add up to the target sum shown in the corner. No digit can repeat within a cage, adding an extra layer of constraint. This ultimate Sudoku variant demands both logical deduction and mental math skills!

🎯 Objectives

Fill the 9×9 grid so each row, column, and 3×3 box contains digits 1-9 exactly once, while also ensuring each cage's numbers sum to its target. Master both Sudoku logic and arithmetic combinations to solve these challenging puzzles!

🎮 How to Play

Each cage shows a sum in the corner. Numbers in each cage must add up to that sum. No number can repeat within a cage. Standard Sudoku rules also apply.

⌨️ Controls

Click a cell to select it, then enter a number 1-9. Use notes mode to pencil in candidates. Wrong entries may be highlighted.

⚙️ Game Mechanics

  • ▸ Standard Sudoku rules: 1-9 in each row, column, and box
  • ▸ Cages (dashed regions) must sum to their target number
  • ▸ No digit can repeat within a single cage
  • ▸ Use the sum constraints to eliminate possibilities
  • ▸ Combine Sudoku logic with arithmetic reasoning

✨ Features

  • ★ Notes Mode - Track candidate numbers
  • ★ Error Detection - Highlight mistakes

💡 Tips

  • ✓ Look for cages with only one possible combination
  • ✓ Small cages (sum 3, 4) are easiest to solve first
  • ✓ Use Sudoku elimination with cage constraints
  • ✓ Track which numbers are used in each cage

❓ Frequently Asked Questions

How is this different from regular Sudoku?
Killer Sudoku adds cage constraints—groups of cells that must sum to a specific total. No digit can repeat within a cage, even if cells are in different rows/columns/boxes.
What's the best starting strategy?
Look for small cages with limited combinations. A '3' cage with 2 cells can only be {1,2}. A '17' cage with 2 cells can only be {8,9}. Start with the most constrained areas.
Can I use normal Sudoku techniques?
Absolutely! All standard Sudoku techniques apply. In fact, you'll need to combine row/column/box elimination with cage constraints to solve these puzzles.
What if a cage spans multiple boxes?
The cage sum constraint still applies across all boxes. However, you still can't repeat digits within the cage, and standard Sudoku rules apply to each box individually.
