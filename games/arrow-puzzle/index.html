Arrow Puzzle Game Guide

📖 About

Arrow Puzzle is a brain-teasing logic game where you must clear a grid of arrows by clicking them in the right order. Each arrow points in one or more directions — click it and it slides along that direction until it exits the grid. But if its path is blocked by another arrow, it bounces back and counts as a mistake! With 200 handcrafted and procedurally generated levels across 4 difficulty chapters, Arrow Puzzle offers endless strategic thinking challenges. Features include multi-directional arrows, number blocks, walls, undo, hints, and star ratings.

🎯 Objectives

Clear all arrows from the grid by clicking them in the correct order. Each arrow slides along its pointing direction and exits the grid if the path is clear. Avoid mistakes by planning your moves carefully — you only get 3 mistakes per level!

🎮 How to Play

Click an arrow to make it slide in its pointing direction. If the path is clear, the arrow exits the grid and disappears. If blocked by another arrow or wall, it bounces back (mistake). Clear all arrows to win. Plan your order carefully!

⌨️ Controls

Click or tap an arrow to activate it. Use the Undo button to reverse your last move. Use the Hint button to highlight the next arrow you should click.

⚙️ Game Mechanics

  • ▸ Arrows slide in their pointing direction until they exit the grid
  • ▸ If an arrow hits another arrow or wall, it bounces back (mistake)
  • ▸ 3 mistakes = game over
  • ▸ Multi-direction arrows must have at least one clear path
  • ▸ Number blocks disappear after being hit N times
  • ▸ Later levels introduce walls and number blocks

✨ Features

  • ★ 200 Levels - Across 4 difficulty chapters
  • ★ Multi-direction Arrows - Complex path planning
  • ★ Number Blocks - Strategic obstacle clearing

💡 Tips

  • ✓ Start with arrows on the edges — they have fewer obstacles
  • ✓ Look for arrows that block the most others, clear those first
  • ✓ Use the Hint button if stuck — it shows the next correct move
  • ✓ Multi-direction arrows are more flexible, save them for later
  • ✓ Aim for 0 mistakes to earn 3 stars
  • ✓ In later levels, number blocks can be used as temporary walls

❓ Frequently Asked Questions

How do I unlock more levels?
Complete levels to unlock the next one. Each chapter has 30-80 levels. Complete earlier chapters to unlock harder ones.
What do multi-direction arrows do?
Multi-direction arrows can exit through any of their pointing directions. At least one path must be clear for the arrow to exit. This gives you more flexibility in planning.
What are number blocks?
Number blocks show a number N. They act as walls until an arrow passes over them N times. After that, they disappear and the cell becomes empty.
How are star ratings calculated?
You earn 3 stars for 0 mistakes, 2 stars for 1 mistake, and 1 star for 2 mistakes. Your best star rating is saved for each level.
