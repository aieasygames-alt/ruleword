Kakurasu Game Guide

📖 About

Kakurasu is the clever Japanese logic puzzle that combines simple addition with logical deduction! A grid of cells awaits your selection, with row and column numbers as clues. When you mark a cell, it contributes its column number to the row sum and its row number to the column sum. Use the target sums to determine which cells must be marked. Simple arithmetic, satisfying logic!

🎯 Objectives

Mark cells in the grid so that each row's sum of column values and each column's sum of row values match the given clues. Every marked cell adds to two different sums—use this constraint to deduce the correct pattern!

🎮 How to Play

Click cells to toggle them. The row clue shows the sum of column values for cells marked in that row. The column clue shows the sum of row values for cells marked in that column.

⌨️ Controls

Click on any cell to mark or unmark it. Marked cells contribute their column number to the row sum and their row number to the column sum. The goal is to match all clue numbers.

⚙️ Game Mechanics

  • ▸ Marked cells add column value to row sum
  • ▸ Marked cells add row value to column sum
  • ▸ Each clue must be matched exactly
  • ▸ Grid typically 5x5 to 10x10
  • ▸ Pure logic - no guessing needed

✨ Features

💡 Tips

❓ Frequently Asked Questions

How do the sums work?
In a 5×5 grid, columns are numbered 1-5 and rows are numbered 1-5. If you mark the cell at row 3, column 4, it adds 4 to row 3's sum and 3 to column 4's sum.
What's the best starting strategy?
Look for rows or columns with clue 0 (no cells marked) or with maximum possible sums (all cells must be marked). These give you free information to start your deductions.
Can I use elimination logic?
Yes! If marking a cell would make a sum impossible, that cell must be unmarked. Similarly, if only one combination of cells can produce a sum, those cells are forced.
Is Kakurasu always solvable without guessing?
Well-designed Kakurasu puzzles have unique solutions solvable through pure logic. Each constraint should lead naturally to the next deduction without trial and error.
